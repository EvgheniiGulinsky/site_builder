# MoreVPN Site

VPN подписочный сервис.

## Разработка

```bash
# Запуск dev сервера
pnpm dev

# Билд
pnpm build

# Запуск production сервера
pnpm start
```

## Конфигурация

- `config/site.config.ts` - основные настройки
- `config/theme.config.ts` - тема (цвета, шрифты)

## URL

- Dev: http://localhost:3000
- Production: https://morevpn.tech
